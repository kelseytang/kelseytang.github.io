Background

Aristotle — a kid-focus smart speaker integrated with smart lights. Among Aristotle's features is Story Time, which lets users play stories and customize settings such as light effects, sound volume, and sleep timer. The project focused on designing user interfaces for the Story Time feature.

My Role in the Project

  1. Created wireframes and user flows
  2. Designed the user interfaces
  3. Built high-fidelity prototypes
